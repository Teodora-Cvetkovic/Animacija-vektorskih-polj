# Animacija-vektorskih-polj

Projekt pri predmetu Matematika z računalnikom na FMF je narejen po vzoru na [Vector field](https://anvaka.github.io/fieldplay/).

# Ideja projekta

Ideja projekta je bila vizualizacija vektorskih polj. Najbolj enostavna opcija je bila, da bi risala puščice v smeri, v kateri bi se delci mogli gibati. Dolžina puščice bi označevala hitrost delca. To sem naredila v *test.py*, ampak je zelo dolgočasno in nezanimivo.
